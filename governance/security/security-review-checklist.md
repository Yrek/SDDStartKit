# Security Review Checklist

Use this when a feature or requirement is security-relevant.

## Checklist
- Does it affect authentication?
- Does it affect authorization or visibility?
- Does it affect tenant separation?
- Does it involve external users or invitations?
- Does it involve file upload or attachment handling?
- Does it affect offline/local storage?
- Does it affect APIs or integrations?
- Does it affect audit logging?
- Does it affect encryption or secrets?
- Does it affect signatures, document integrity, or evidence value?
- Does it change export/report visibility rules?
- Does it require architecture review before implementation?
